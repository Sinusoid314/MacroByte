LOC(handle)


Description:

Returns the current read/write position of an open file specified by handle. This function is intended for files opened in binary mode.


Usage:

showconsole

var entry as string

open "test.txt" for output as #1
print #1, "One,Two"
print #1, "Three,Four"
close #1

open "test.txt" for binary as #1
input #1, entry
print entry
print LOC(1)
close #1

Produces:
One
4
