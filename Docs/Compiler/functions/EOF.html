EOF(handle)


Description:

Returns -1 if the end of the file specified by handle has been reached, or 0 if it hasn't.


Usage:

showconsole

var entry as string

open "test.txt" for output as #1
print #1, "One,Two"
print #1, "Three,Four"
close #1

open "test.txt" for input as #1
while EOF(1) = 0
   input #1, entry
   print entry
wend
close #1

Produces:
One
Two
Three
Four
