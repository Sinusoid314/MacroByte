INPUT(handle, length)


Description:

Returns a string containing a section of bytes from the file handle. The number of bytes to retrieve from the file is specified by length.


Usage:

var n as number

showconsole

open "test.txt" for output as #1
print #1, "One,Two"
print #1, "Three,Four"
close #1

open "test.txt" for input as #1
for n = 1 to lof(1)
   print Input(1, 1);
next n
close #1

Produces:
One,Two
Three,Four
