INPUT(#handle, length)


Description:

Returns a string containing a section of bytes from the file #handle. The number of bytes to retrieve from the file is specified by length.


Usage:

var n as number
var fn as string

showconsol

fn = fileopen("Open...","Text Files (*.txt) | *.txt")

open fn for input as #file

for n = 1 to lof(#file)
     print Input(#file, 1)
next n

close #file
