FOR var = num1 TO num2 [STEP num3]
     {code}
NEXT var


Description:

Allows a program to execute some code a certain amount of times. When the loop begins, the variable specified by var is set to the number specified by num1. Each time the loop runs through, the value of var will be increased by 1, and the loop will end when the value of var is greater than num2. If STEP is specified, then the value of var will be increased by num3 each time the loop runs. The NEXT statement marks the end of the code to be looped, and must be followed by the same var specified after the FOR statement.


Usage:

showconsol
var n as number
For n = 1 To 10
     print n
Next n

2
3
