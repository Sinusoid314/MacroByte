CALL subName arg1, arg2, ...


Description:

Runs a Sub specified by subName with the appropriate arguments. The number of arguments must match up with the number of parameters of the sub being called. The data types of each argument must also match up with the types of their corisponding parameter.


Usage:

showconsole
var nameStr, descStr as string
input "Enter a name: ", nameStr
input "Enter a description: ", descStr
Call Describe nameStr, descStr

Sub Describe(name As String, desc As String)
     print
     print name & " is " & desc
     print
End Sub
