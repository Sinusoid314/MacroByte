RETURN


Description:

Returns code execution to the last GOSUB line that was executed. If no GOSUB command was executed, an error will occur.


Usage:

showconsol
var n as number

for n = 1 to 3
     gosub @printMulti
next n

end

@printMulti
print n*1
print n*2
print n*3
print
Return


Produces:
1
2
3

2
4
6

3
6
9
