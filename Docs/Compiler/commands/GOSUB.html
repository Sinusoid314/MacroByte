GOSUB @branchLabel


Description:

Jumps to a section of code defined by @branchLabel, like the GOTO command, except the flow of control can jump back when a RETURN statement is reached.


Usage:

showconsole
var n as number

for n = 1 to 3
     GOSUB @printMulti
next n

end

@printMulti
print n*1
print n*2
print n*3
print
return


Produces:
1
2
3

2
4
6

3
6
9
