RedimAdd arrayName, itemVal [, beforeIndex ]


Description:

Adds itemVal to the array given by arrayName before the array item given by beforeIndex. If beforeIndex is not specified, itemVal is added to the end of the array. This command only works on one-dimensional arrays.


Usage:

var n as number
array names(3) as string

showconsole

names(0) = "Alice"
names(1) = "Chris"
names(2) = "Dan"

RedimAdd names, "Bob", 1

for n = 0 to 3
   print names(n)
next n

Produces:
Alice
Bob
Chris
Dan
