DIM arrayName(dimSize1, dimSize2, ...) AS type


Description:

Creates an array with the given dimensions. Each dimSize specifies the total number of elements for that dimension of the array. type can be either Number or String depending on what type you want all the values of the array to be.


Usage:

showconsole
Dim names(3) As String
names(0) = "Bebop"
names(1) = "Rocksteady"
names(2) = "Krang"
print names(1)

Produces:
Rocksteady
