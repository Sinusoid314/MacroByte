DIM arrayName(maxIdx1 [,maxIdx2]) AS type


Description:

Creates an array whose lower boundry is 0 and whose upper boundry is specified by maxIdx1. Specify maxIdx2 to create a double-dimensioned array. type can be either Number or String depending on what type you want all the values of the array to be.


Usage:

showconsol
Dim names(3) As String
names(0) = "Bebop"
names(1) = "Rocksteady"
names(2) = "Krang"
print names(1)

Produces:
Rocksteady
