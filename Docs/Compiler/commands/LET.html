LET varName = expression


Description:

Assigns the value of expression to the variable specified by varName. The expression type must match the variable type (i.e. a string variable can only be assigned a string expression.) The LET command is optional, though, and assignments can be made without having the LET statement before them.


Usage:

showconsole

var cv as string
var av as number

input "Enter a character: ", cv
LET av = asc(cv)
print av
