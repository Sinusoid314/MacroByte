Line Input #handle, variable


Description:

Reads in a line of data from the file specified by #handle and assigns it to variable. A line of data is defined as being terminated by both a carriage return and newline character together.


Usage:

showconsole
var entry as string

open "file.txt" for output as #1
print #1, "One, Two"
print #1, "Three, Four"
close #1

open "file.txt" for input as #1
Line Input #1, entry
close #1

print entry

Produces:
One, Two
