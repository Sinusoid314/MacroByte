INPUT prompt, variable

~OR~
INPUT #handle, variable

Description:

Takes in a value from either a file or the consol. If prompt and variable are specified, the prompt expression is evaluated and printed onto the consol window and the consol waits for the user to enter something. If prompt is omitted, then the consol waits for input without printing a prompt.

If #handle and variable are specified instead, a value is read into variable from the file specified by #handle.


Usage:

showconsol
var entry as string

Input "Enter text to print to file: ", entry

open "file.txt" for output as #1
print #1, entry
close #1

open "file.txt" for input as #1
Input #1, entry
close #1

print entry
