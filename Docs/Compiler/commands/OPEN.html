OPEN fileName FOR mode AS #handle


Description:

Opens a file specified by fileName. The file is given a #handle to use when reading or writing to it. Close the file with the CLOSE command. There are four different modes the file can be opened in:

input - Opens the file for reading. The file specified by fileName must already exist.
output - Opens the file for writting to. If the file already exists, the data in it is erased. Otherwise, a new file is created to be written to.
append - Opens the file for writting to, like output, but does not erase the file's data if it already exist.
binary - Opens a file for both reading and writting. This mode works with the file byte by byte, ignoring line delimiters.


Usage:

showconsole
var fileDat as string

input "Enter file data: ", fileDat

Open "c:\file.dat" For output As #1
     print #1, fileDat
close #1
