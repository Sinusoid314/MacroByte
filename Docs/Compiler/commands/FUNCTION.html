FUNCTION funcName(param1 AS type, param2 AS type, ...) AS type
     {code}
END FUNCTION


Description:

Functions are much like Subs, with the exception that functions return a value, whose type is specified by the type at the end of the parameter list (either String or Number). A function can have any number of parameters, each of which is specified inside the parenthesis much like a varible declaration in the VAR command. To return a value from a function, just set the value of the variable with the same name of the function (which is specified by funcName). Like subs, all variables and arrays declared within functions can only be recognized and used within the function they are declared. Also like subs, functions can be placed anywhere in the program code except for other function or sub definitions.

     var newNum as number
