ONERROR subName


Description:

Calls the Sub specified by subName when a runtime error occurs, instead of displaying an error message. subName must have either no parameters, or one string parameter to recieve the error message text.


Usage:

array testArr(3) as string

OnError CatchError

testArr(5) = "New string"
end

sub CatchError(errorMsg as string)
     message errorMsg, "CatchError"
end sub
