IF expression THEN {command}

~OR~
IF expression THEN
          {code}
     ELSEIF expression THEN
          {code}
     ELSE
          {code}
END IF
Description:

Executes a certain block of code based on the given expressions. The first form accepts an expression that is to be evaluated. If the expression is true, the {command} after it will be executed, otherwise the line is skipped. The second form allows for multiple lines of code in one block. If the expression after the IF statement is true, the {code} after it is executed. If not, the ELSEIF expressions are evaluated. There is no limit to the amount of ELSEIF statements that can be in one IF block. The {code} after the first true ELSEIF expression will be executed. If none of the ELSEIF expressions are true, the {code} after the ELSE statement will be executed. The END IF statement marks the end of the IF block.


Usage:

showconsol
var num as number
input "Enter a number: ", num
If num < 10 Then
     print "Number is less than 10"
ElseIf num > 10 Then
     print "Number is greater than 10"
Else
     print "Number is 10"
End If
