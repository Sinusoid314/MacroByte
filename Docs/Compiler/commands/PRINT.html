PRINT [#handle,] [expression]


Description:

Prints data to the consol window or a file. If #handle is specified, then expression is evaluated and written to the file, and must also be specified. If only expression is specified, then that expression will be evaluated and printed to the consol window. If neither are specified, then a blank line is printed to the consol window. Placing a ; at the end of the entire PRINT statement will keep it from printing a carriage return to the consol window or file.


Usage:

showconsol

Print "Hello, ";
Print "world!"
Print

open "file.txt" for output as #1
Print #1, "stuff"
close #1
