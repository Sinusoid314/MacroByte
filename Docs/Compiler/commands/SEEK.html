SEEK #handle, position


Description:

Sets the read/write position of the file specified by #handle. When data is printed to or read from the file, it will begin from this byte position.


Usage:

var n as number

open "testFile.txt" for binary as #1

print #1, "123456789"

for n = 2 to 8 step 2
     Seek #1, n
     print #1, "*";
next n

close #1
