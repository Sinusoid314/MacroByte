RedimRemove arrayName, itemIndex


Description:

Removes the item at itemIndex from the array given by arrayName. This command only works on one-dimensional arrays.


Usage:

var n as number
array names(4) as string

showconsole

names(0) = "Alice"
names(1) = "Bob"
names(2) = "Chris"
names(3) = "Dan"

RedimRemove names, 1

for n = 0 to 2
   print names(n)
next n

Produces:
Alice
Chris
Dan
